\vskip命令是在命令出现时立马生成垂直间距；生成间距后，开始一个新的段落（逻辑上分段）。

        outPut.text = String(result)

    ope1 = ope1 + value!
        outPut.text = ope1

Matlab 矩阵计算

计算矩阵方差
diag()

diag(X,k);

若 X 是矩阵，则 diag(X) 为 X 的主对角线向量

若 X 是向量，diag(X) 产生以 X 为主对角线的对角矩阵

C = cov(x.^2);

得到的是协方差矩阵. covariance而不是variance.

C = cov(x.^2,1);

为计算方差。

C = cov(x.^2); % square covariance

C = C ./ sqrt(diag(C) * diag(C)’); % square correlation

引用比喻:
一行为一个样本，计算列之间的相似度。
每一行为一个男生，列表示男生的性格，受女生程度等。
性格和女生受欢迎程度的协方差。
设x是1016维，cov是1616维，C是1616维
diag(C)是C的主对角线，diag(C).diag(C)’ 161116，自相关度高的点值大。
1616除以1616，得到16*16，主对角线，自相关为1.

cov(x)
求向量x的方差。
cov(x)为一个数值，数值大小计算公式为S(x)。
cov(x,y)
求向量x与y的协方差矩阵。
cov(x,y)为2*2矩阵，
[S(x)C(x,y);
C(y,x)S(y);]

为区别对待，
cov(x)又记作cov(x,0)
cov(x,y)又记作cov(x,y,0)
cov(X) 又记作cov(X,0)
cov(X,Y) 又记作cov(x,y,0)
对于归一化参数为1/(N-1)的情况，当N=1时，自动将参数调整为1/N。

im2col 获得子矩阵
im2col(reshape(X(i,1:1024),CIFAR_DIM(1:2)), [rfSize rfSize]) ;
结果是729108(经过一次转置)，每一行保存一个patch，

函数原型：B = im2col(A,[m n],block_type)
